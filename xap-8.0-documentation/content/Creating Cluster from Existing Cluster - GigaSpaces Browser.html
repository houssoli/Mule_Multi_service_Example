Creating Cluster from Existing Cluster - GigaSpaces Browser


Section Summary: Using an existing cluster as a basis for a new cluster.

Overview

You can create a new cluster configuration file using the existing one as a basis.

  1. Click the Cluster menu and choose New From. The following window will appear:
  2. Select the existing cluster configuration file and click Open. The GigaSpaces Cluster Wizard will open. Specify a new cluster name and a different cluster configuration file name. Click Next. Now you will be able to edit a new cluster configuration based on the existing file.
