UI Undeploy and Update Options


Section Summary: UI Undeploy, Redeploy, and Update options.

Overview

  • Undeploy - right-clicking the deployed service and selecting Undeploy shuts down the service.


  • Update - right clicking the deployed service and selecting Update allows you to make incremental changes to your Deployment Descriptor and then dynamically update it.

    When using the Update option, some of your changes might not be taken into account. It is recommended to use Redeploy instead.
