Verifying Cluster Configuration - GigaSpaces Browser


Section Summary: Verifying the cluster configuration is OK and placing it on member machines.

Overview

  1. Click Next to go to the Verify Cluster window. If there are no problems with your cluster group definitions, you will see the following:
  2. You can preview the cluster configuration file by clicking View File:
  3. Click Close to exit the preview window and then Finish on the Verify Cluster window.
  4. You will need to place the cluster file location:
  5. Click Save.
    You now have a cluster configuration file and are ready to create the cluster.
  6. Click the Space icon in the Container Tree. The space properties will be displayed in the right hand panel. Check the Clustered Space box and type in the cluster configuration file created earlier.
  7. Click Update to apply the changes.
    You should repeat this task for every space member in the cluster.
  8. Restart the GigaSpaces Server to refresh its memory with all the new definitions.
