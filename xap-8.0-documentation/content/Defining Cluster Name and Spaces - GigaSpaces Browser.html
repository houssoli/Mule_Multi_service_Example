Defining Cluster Name and Spaces - GigaSpaces Browser


Section Summary: Defining the cluster name and spaces intended to be members in the static cluster.

Overview

The first step in creating a cluster is to define its name, specify the cluster configuration file, and plan for the spaces in it.
To define a cluster and its spaces:

  1. Start the GigaSpaces Browser.
  2. In the Cluster menu, click New. The Create a New Cluster dialog appears.

  3. Use the Cluster Name field to enter a name for the cluster, which is used in the GigaSpaces environment.
  4. In the Cluster File field, type the path to the cluster configuration file, and its name. The cluster configuration file does not exist yet – this specifies the path it is created in, and its filename. Take note of this path and name, as you have to specify it when creating the cluster's spaces.
  5. Click the Next button to continue. The Define the Cluster Space-Members dialog box is displayed.
    In this step you plan the cluster by defining which spaces should be members. You do not actually create the spaces here – this is done after you finish the Cluster Wizard.
  6. Click the Add button. The Setup the Cluster Members Attributes dialog box is displayed.
  7. Press the ... button to the right of the Member Name field. The URL Setting... dialog opens.
  8. Type the Host Name, Container Name and Space Name for the space you wish to create and press OK. In the Setup Cluster Members Attributes dialog, the Member Name and Member URL fields are automatically filled in.
  9. Click the Create button to define the space and return to the Define the Cluster Space-Members dialog box.
  10. Repeat steps 6-9 to define additional spaces. You should define at least two spaces.
  11. The spaces you defined are added to the Member Name list in the *Define the Cluster Space-Members dialog box.
    The spaces you have defined need not necessarily exist. The system does not check the paths and names of the spaces you specify. It is assumed that you use the same paths and names to create the space in a subsequent procedure.
  12. Create cluster groups by following the procedure in the following section.
