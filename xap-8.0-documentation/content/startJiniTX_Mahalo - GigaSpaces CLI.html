startJiniTX_Mahalo - GigaSpaces CLI


Section Summary: Starts the Jini Transaction Manager as a stand alone process.

Syntax

startJiniTX_Mahalo

Description

The startJiniTX_Mahalo starts the Jini Transaction Manager as a stand alone process.

For details on how to start an embedded Jini Transaction Manager, refer to the gsInstance - GigaSpaces CLI section.

Options

None.

Example

<GigaSpaces Root>\bin>startJiniTX_Mahalo.bat
