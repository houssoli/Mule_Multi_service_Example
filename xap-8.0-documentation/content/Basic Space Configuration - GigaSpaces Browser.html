Basic Space Configuration - GigaSpaces Browser


Section Summary: Using the basic space configuration dialog.

Overview

The Standard mode of the Create New Space in Container dialog allows you to view or modify basic space settings.

The Persistent Options area is only relevant if the Persistent check-box in the Create New Space dialog is checked.

The Cluster Options area is only relevant if the Clustered check-box at the top of the Create New Space in Container dialog is checked.
Cluster Config URL – the URL of the configuration file of the cluster to which this space should belong. If you have created a cluster that refers to this space, you must enter the URL of the cluster configuration file here (for every space in the cluster). Another option is to use user defined clustered space space.
